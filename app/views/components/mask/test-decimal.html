Mask Test: Edit decimal value a second time ignoring decimal point

1. Enter the value .8
2. Blur out of the field, see that the value in it is 0.8
3. Highlight the value and try and enter the value .7
4. See that the . is ignored and that the value is 7 when it should be .7

Note: using allow leading zeros allows you to enter 0.7 in step 3, which has the decimal point maintained
